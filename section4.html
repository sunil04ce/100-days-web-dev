HTML & CSS Basics Summary

HTML Summary

HTML (HyperText Markup Language) is the code that is used to structure a web page and its content. You don't use it for styling purposes. Use CSS for that instead!

  • HTML uses "elements" to describe (annotate) content
  • HTML elements typically have an opening
  • You can also have void(empty) elements like images

Learn more about all available HTML elements on the MDN HTML element reference

CSS Summary

CSS to style document

You can asssign styles via the 'style' attribute

  • To avoid code duplication, you should use global styling
  • When working with CSS, concepts like 'inheritance', 'specificity', 'box-model' should be understood

Learn more about all available CSS elements on the MDN CSS element reference
